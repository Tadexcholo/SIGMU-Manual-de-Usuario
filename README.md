# SIGMU: Manual de Usuario
> **Sistema Integral de Gestión de Reportes Municipales Urbanos**

Este repositorio contiene la documentación oficial para el usuario del sistema, desarrollado para el Honorable Ayuntamiento de Allende, Coahuila, como parte del proyecto de estadías profesionales del alumno Jose Gustavo Torres Sotelo de la Universidad Tecnológica del Norte de Coahuila.

El archivo principal de este repositorio es el **Manual de Usuario**, el cual detalla el funcionamiento técnico y operativo de la plataforma, incluyendo:

*   Gestión de accesos.
*   Uso de la geolocalización.
*   Importación de reportes desde WhatsApp.
*   Seguimiento de estatus y carga de evidencias fotográficas.
*   Interpretación de gráficas estadísticas.

Puedes visualizar o descargar el manual completo aquí:
👉 [**Manual_de_Usuario_SIGMU.pdf**](./Manual_de_Usuario_SIGMU.pdf)


SIGMU es un sistema tipo **GMAO** (Gestión de Mantenimiento Asistido por Ordenador) desarrollado en PHP y MySQL. Su objetivo es centralizar y optimizar la comunicación entre el departamento de Atención Ciudadana y las áreas operativas del municipio mediante el uso inteligente de geodatos.
